## Ejecutar en desarrollo

1. Clonar repositorio
2. ejecutar
3. ```
   npm install

   ```

4. Tener Nest CLI instalado

   ```
   npm i -g @nestjs/cli

   ```

5. Levantar la base de datos

   ```
   docker compose up -d

   ```

6.

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
